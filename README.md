# Conduit Regression Test Suite

This test suite covers following tests:

1) API Test
2) UI Integration Test
3) E2E Test

To run these tests 
```
git clone

cd Conduit_regression_suite
```
 To open cypress test runner:
 ```
 npx cypress open
```

 To run cypress test runner in a headless mode:
 ```
npx cypress run
```
